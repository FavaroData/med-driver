MEDDRIVE PRINTER — GUIA RÁPIDO
================================

INSTALAÇÃO
----------
Execute o instalador como Administrador:
  - Windows 10/11: MeddrivePrinter-Setup.exe
  - Windows 7:     MeddrivePrinter-Win7-Setup.exe

Após a instalação, abra o MedDriveManager pelo menu Iniciar para
criar perfis de saída e adicionar impressoras virtuais.


PROBLEMA: EXECUÇÃO DE SCRIPTS DESABILITADA
-------------------------------------------
Se aparecer o erro:
  "nao pode ser carregado porque a execucao de scripts foi desabilitada"

Execute no PowerShell (como Administrador):
  Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser


VERIFICAR INSTALAÇÃO
--------------------
Para confirmar que o driver foi instalado corretamente, execute no PowerShell:

  Get-Printer | Where-Object { $_.PortName -like "Meddrive*" } |
      Select-Object Name, DriverName, PortName

Ou use o script de diagnóstico completo (raiz do repositório):
  .\tests\diagnostico.ps1


LOGS
----
  Instalação:  C:\Windows\Temp\meddrive_install.log
  Impressão:   C:\Windows\Temp\meddrivemon_init.log


ORDEM DE USO
------------
1. Instalar o MeddrivePrinter-Setup.exe (ou Win7)
2. Abrir o MedDriveManager
3. Criar um perfil de saída (aba Perfis)
4. Adicionar uma impressora vinculada ao perfil (aba Impressoras)
5. Imprimir a partir de qualquer aplicativo selecionando a impressora criada
